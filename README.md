# tt-database
Análisis de confiabilidad de tiempos de viaje: Este repositorio contrasta los resultados del software StreetVal de HCM con datos obtenidos mediante un algoritmo propio de extracción en Google Maps. Estudio de caso en Morelia, México, evaluando métricas PTI, BI y TTI para la calibración de modelos de transporte en contextos urbanos emergentes.
